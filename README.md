## 快速开始
### 安装
```bash
git clone https://github.com/你的账号/diabetes-project
pip install -r requirements.txt  # 只有numpy和scikit-learn两个依赖
```

### 使用示例
```bash
# 输入：血糖值  BMI  年龄
python predict.py 148 33.6 33
```

### 输出解释
```

说明：您的血糖值148mg/dL超过正常范围(参考WHO标准)
```
